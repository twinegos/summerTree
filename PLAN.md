# summerTree — 개발 계획서 v1.0

## 프로젝트 개요

식물 가게용 모바일 웹 앱.
- 식물마다 QR 코드 부착 → 고객이 핸드폰으로 스캔 → 식물 상세 페이지
- 식물 설명, 키우는 방법, 주의사항, 가격 표시
- 장바구니, 결제, 관리자 페이지

## 기술 스택

- 프레임워크: Nuxt 3 (Vue 3) — Qt 위젯 구조와 유사, .vue 파일 = 위젯 파일
- 데이터베이스: Supabase (PostgreSQL) — 웹 대시보드 관리, Storage, Auth 내장
- 결제: 포트원 V2 + 토스페이먼츠 (기본) + 카카오페이 (추가)
- QR 코드: qrcode.js — 식물별 자동 생성, 인쇄용 PDF 출력
- 배포: Vercel
- 스타일: Tailwind CSS

## 에이전트 워크플로우

- Claude Code (주 개발): 컴포넌트, API, DB 스키마 구현
- Hermes Agent (자동화 보조):
  - 서브에이전트로 프론트/백엔드 병렬 분업
  - Docker 백엔드로 결제 웹훅 격리 테스트
  - Cron 야간 자동 테스트
